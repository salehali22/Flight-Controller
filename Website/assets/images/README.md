# Images Directory

Place your project images here.

## Recommended Structure

```
assets/images/
├── gallery/
│   ├── pcb-design.jpg
│   ├── component-assembly.jpg
│   ├── flight-testing.jpg
│   └── schematic-diagrams.jpg
├── team/
│   ├── akaki.jpg
│   ├── saleh.jpg
│   └── levani.jpg
└── hardware/
    ├── pcb-top.jpg
    ├── pcb-bottom.jpg
    └── assembled-board.jpg
```

## Image Recommendations

### Gallery Images
- **Dimensions**: 1920x1080px (16:9 ratio)
- **Format**: JPG or PNG
- **File Size**: < 500KB (optimized)

### Team Photos
- **Dimensions**: 500x500px minimum (square)
- **Format**: JPG or PNG
- **File Size**: < 200KB

### Hardware Photos
- **Dimensions**: 1920x1080px or higher
- **Format**: PNG for screenshots, JPG for photos
- **File Size**: < 1MB

## Adding Images

1. Add image files to this directory
2. Update `index.html` to reference the images
3. Use relative paths: `assets/images/your-image.jpg`

## Image Optimization

Use online tools to optimize images:
- TinyPNG (tinypng.com) - PNG/JPG compression
- Squoosh (squoosh.app) - Image optimization
- ImageOptim - Batch optimization
